# Portfolio Demos

This folder contains three GitHub-ready portfolio demos that showcase practical automation work:

1. `ecommerce-scraper-demo`
   A Python product-listing scraper with normalization, deduplication, and CSV export.
2. `excel-report-automation-demo`
   A one-click weekly reporting workflow that cleans messy operational data and generates an Excel workbook.
3. `crm-api-sync-demo`
   A CRM-style API sync demo with pagination, upserts, logging, and SQLite storage.

These are sample portfolio projects, not client work.
